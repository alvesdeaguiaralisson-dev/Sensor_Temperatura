# 🌡️ Sensor de Temperatura com LEDs (Arduino)

## 📌 Descrição
Projeto desenvolvido para monitorar a temperatura e representar visualmente sua variação utilizando LEDs. Conforme a temperatura aumenta, mais LEDs são acionados; conforme diminui, menos LEDs permanecem acesos.

## ⚙️ Tecnologias utilizadas
- Arduino
- Sensor de temperatura (ex: TMP36 ou LM35)
- LEDs
- Resistores

## 🚀 Funcionamento
O sistema realiza a leitura da temperatura por meio de um sensor analógico conectado ao Arduino. Com base no valor obtido, o código define faixas de temperatura e aciona uma quantidade proporcional de LEDs, criando uma indicação visual simples e intuitiva.
